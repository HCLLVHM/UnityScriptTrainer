# 镇邪 Zhenxie 内置修改器 

### 安装方法
- 先安装 [BepInEx](https://github.com/BepInEx/BepInEx/releases/tag/v5.4.5 "BepInEx")
- 然后将ScriptTrainer.dll放到 镇邪 Zhenxie\ZX30\BepInEx\plugins中；
- 如果没有“plugins”文件夹自己创建一个
- 进游戏按F9呼出修改器菜单

### 功能
- 回满血量，蓝量
- 无敌
- 技能点修改
- 获取道具
